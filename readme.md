## Installation

Available on Packagist ([ttek/tk-mvc](http://packagist.org/packages/ttek/tk-mvc))
and installable via [Composer](http://getcomposer.org/).

```bash
composer require ttek/tk-mvc
```

Or add the following to your composer.json file:

```json
"ttek/tk-mvc": "~2.0"
```
